Em computação os dados podem ser organizados em várias formas, o modelo lógico ou matemático de organização particular de dados denomina-se Estrutura de Dados.

Leia atentamente o enunciado que se segue e responda as questões:

1. Refira-se às características das Estruturas de Dados e Algorítmos

2. Refira-se a classificação das Estruturas de Dados e dê exemplos para 2 tipos de estruturas.

3. Conceitue complexidade de algorítmos e discuta a sua importância na performance de algorítmos.

4. Prove que f(n) está em O(n^2) para n >= 4

5. Critique a seguinte definição alternativa da classe O: Dizemos que f está em O(g) se existem números positivos c, n inicial e n>=n inicial tais que f(n)<= c g(n).

6. Critique a seguinte definição alternativa da classe O: Dizemos que f está em O(g) se existe um número natural (n incial) tal que f(n) <= c g(n) para algum número positivo c e para todo n>=n inicial.

7. Verifique se 4n^2 + 2n é o(9n^2).

8. É verdade que n^2 + 200n + 300 = O(n^2)?

9. É verdade que 2n+1 está em O(2^n)? É verdade que 3n está em O(2^n)?

10. Dado o seguinte fragmento de um programa, encontre a complexidade de: pior caso, melhor caso e caso médio, isto é, calcule Big O, Omega e Teta.

int factorial (int n)
{
int i, fact=1;
for (i=1; i<=n; i++)
fact *=i;
return fact;
}

11. Escreva um algorítmo para calcular o dobro de um número natural e calcule a complexidade de melhor caso.
