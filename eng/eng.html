Essential Basics

Affirmative Essential Structures

This is a man

This is a woman

This is a boy

This is a girl

This is a motor car

Interrogative Essential Structures

Is this a tree?

Yes, this is a tree.

Is this a ship?

Yes, it is.

Negative Essential Structures

Is this a mouse?

No, it is not a mouse. It is a horse.

How to use an?

We use a berfore a consonant sound. We use an before a vowel sound.

It is an inkpot.

It is an ice-cream.

Is this an apple? Yes, it is.

Is this an umbrella or an aeroplane? Yes, it is (it's) an aeroplane.

How to use in?

The egg is in the egg-cup.

The girl is in the classroom.

How to use where?

Where is (where's) the boy? The boy is in the bed.

Where is the train? The train is in the station.

How to use on?

The ship is on the sea.

The clock is on the wall.

The apple is on plate.

How to use under?

The cat is under the umbrella.

What is under the table? The mouse is.

Is the train in the sky? No, it is not (isn't).

Is the dog under the umbrella? No, it is not.

There is and there are

There is one boy | There are three boys.

There is one tree | There are five trees.

There is one shilling | There are two shillings.

There is one man | There are two men.

There is one woman | There are three women.

There is one child | There are three children.

These is and these are

This is a football. These are footballs.

This is a king. These are kings.

This is a child. These are children.

What are this? These are cups and saucers.

Are these women? No, these are not women. These are children.

That is and those are

That is the sun. It is not near the earth. It is far away.

Those are stars. The stars are not near the earth. The stars are far away.

Is that the moon? Yes, that is the moon.

Is there | are there?

There is a boy in the picture.

Is there an aeroplane in this picture? Yes, there is.

There are footballs in this picture.

Are there flowers in this picture? No, there are not.

Singular | Plural

Singular + s = Plural. Exceptions: man-men, woman-women, child-children.

The child. The children.

A man. Three men.

A window. Four Windows.

The window. The windows.

The tree. The trees.
